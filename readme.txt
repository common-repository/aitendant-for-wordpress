= Make Customers Everywhere =

aitendant help retailers and fundraisers expand their sales and service footprint by enabling their users to make purchases and manage their account even while visiting external websites/blogs. Our big idea is that aitendant empowers you to help your customers anywhere across the web instead of forcing them to a single location. Think of aitendant as a store that activates everywhere you have a presence.

aitendant for WordPress adds the necessary script to enable aitendant on any WordPress blog. This eliminates the need to edit your template code to begin using aitendant. 

= Features =

aitendant for WordPress has the following features:

* Simple and intuitive interface.
* Automatically insert aitendant eCommerce platform in your website/blog.
* Sell products and manage customers across external websites/blogs.
* aitendant works with your current content, no changes necessary.
* Fully customizable interface to match your website/blog. 

For more information, visit the [aitendant for WordPress plugin page](http://www.aitendant.com/plugins/wordpress/aitendant/).

== Installation ==

1.  Download aitendant for WordPress plugin file.
2.  Unzip and upload the plugin file folder to your "/wp-content/plugins/" folder or install by using WordPress' built-in installer.
3. Activate the plugin through the "Plugins" menu in WordPress.
4. Thats it!

== Frequently Asked Questions ==

= If I already have jQuery installed on my blog will installing aitendant for WordPress conflict? =

No.  aitendant for WordPress is designed not to conflict if you have jQuery already installed.

= Can I use aitendant without installing the plugin? =

Yes.  aitendant provides an activation link that can be manually embedded on each page you want aitendant enabled.  We recommend that you paste the activation link just before the closing body tag.
